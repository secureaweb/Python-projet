# Python-projet
🔍 Mini Web Scanner Python

📌 Description :
Petit outil de cybersécurité en Python permettant d’envoyer une requête HTTP vers une URL et d’analyser le code de réponse du serveur.

🎯 Objectif :
Comprendre les requêtes HTTP
Analyser les codes de statut HTTP
Découvrir les bases de la reconnaissance web

💻 Technologies
Python
requests

▶️ Utilisation
python main.py


📊 Exemple de résultat
URL : https://secureaweb.fr
Status code : 403


🧠 Interprétation
200 → OK
3xx → redirection
4xx → erreur client (ex : 403 Forbidden)
5xx → erreur serveur

⚠️ Note
Projet réalisé dans un cadre d’apprentissage en cybersécurité, uniquement sur des cibles autorisées (mon propre site web).
